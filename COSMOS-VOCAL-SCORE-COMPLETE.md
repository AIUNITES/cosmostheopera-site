## Vocal Resonance Notation (VRN)

### Primary Resonance Areas
| Code | Name | Description |
|------|------|-------------|
| [C] | Chest | Resonance in chest cavity |
| [L] | Larynx | Centered around voice box |
| [P] | Pharynx | Throat area resonance |
| [O] | Oral | Mouth and oral cavity |
| [N] | Nasal | Nasal cavity resonance |
| [H] | Head | Encompasses nasal, oral, sinus |

### Additional Indicators
| Code | Name | Description |
|------|------|-------------|
| [T] | Thyroid | Vibration in thyroid area |
| [Tg] | Tongue | Position affecting resonance |
| [SP] | Soft Palate | Position affecting balance |
| [M] | Musculo | Overall muscle engagement |
| [Oc] | Occipital | Back of head resonance |
| [Ey] | Eye Muscles | Mask resonance engagement |
| [B] | Brow | Brow muscle engagement |
| [Vf] | Ventricular Folds | False fold engagement |
| [Flt] | Falsetto | Falsetto mechanism |
| [Wr] | Whistle | Whistle register |

### Degree Symbols
- **+** = slight intensity
- **++** = moderate intensity
- **+++** = strong intensity
- **-** = decreased/reduced
- **(R)** = relaxed
- **(E)** = engaged
- **(A)** = arched
- **(F)** = forward
